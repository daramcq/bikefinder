Setup Guidelines

1) Download the Python Installer
   Windows: https://www.python.org/ftp/python/2.7.8/python-2.7.8.msi
   Mac: https://www.python.org/ftp/python/2.7.8/python-2.7.8-macosx10.6.dmg

2) Double click the Python installer and follow the guidelines

3) Open Python IDLE

4) Click File -> Open File (hacks_hackers/bikefinder.py)

5) From bikefinder window, click Run -> Run Module
